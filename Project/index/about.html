This simple website was designed by Ban Abdul Ridha as a member of the Code For IRAQ initiative , Code For IRAQ is a humanitarian, non-profit initiative that aims to serve the community through programming. Programming by an Iraqi programmer Muhammad Issa.

Code For IRAQ is a real educational initiative that takes care of those interested in learning to design and program mobile applications, websites, computer programs, networks, communications and computer operating systems using open source technologies, and provides them with all the necessary educational lessons completely free of charge.
